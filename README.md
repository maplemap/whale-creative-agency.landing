# Whale-Design landing-page application

[Live Demo](https://maplemap.github.io/whale-creative-agency-page/)

### Installation (development)
1. clone or download project
2. ```npm install```
3. ```npm run devserver```
4. open http://localhost:8090

### Installation (test)
1. clone or download project
2. ```cd public/```
3. run index.html
